# PV-Operations-Dashboard
Power BI dashboard for monitoring Pharmacovigilance operational KPIs
Dashboard Preview

Key Metrics

Total Case Volume
Serious vs Non-Serious Cases
Workflow Distribution
Case Processing Trend
Drill throughs
Priority Case Monitoring

Tools Used
Power BI
DAX
Data Modeling
Pharmacovigilance Analytics

Business Problem

Pharmacovigilance teams process large volumes of safety cases daily.
Tracking case seriousness, Due today cases, processing workload, and workflow distribution is critical to ensure regulatory compliance and operational efficiency.
This dashboard helps monitor key operational KPIs and identify workload trends across safety cases.
